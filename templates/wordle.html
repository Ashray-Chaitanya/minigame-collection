Stage 1: Wordle

Objective:
Guess the hidden 5-letter word within 6 attempts.
Gameplay:
The player inputs a 5-letter word guess in each round. After each guess, the game provides feedback on the letters of the guessed word.
Feedback Explanation:
Green : If a letter is correct and in the correct position, that letter's box will turn green.
Yellow : If a letter is correct but in the wrong position, that letter's box will turn yellow.
Gray : If a letter is not in the word at all, that letter's box will turn gray.
Input Rules:
Each guess must be a valid 5-letter word. The game will not accept non-words or words of incorrect length. Words cannot contain numbers, symbols, or special characters.
Winning:
You win if you guess the correct word within 6 attempts. The game ends after a correct guess, proceeding you to the next stage.
